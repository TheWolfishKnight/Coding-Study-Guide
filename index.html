Git

  • git status: checks current working branch.
  • git checkout -b branch_name: Creates a new branch and switches to it automatically.
  • git checkout branch_name: Moves you to branch that is named.
  • git add -A: this commits ALL files in a working branch.
  • (git commit) -m "Message": This will associate a message with the commit. Useful to give a short description of what you did.
  • git pull origin main: Good idea to run this command to check that everything is up to date with the main branch before making a pull request.
